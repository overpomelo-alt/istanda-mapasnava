把「Istanda」加到手機桌面

加到桌面後,點圖示就能直接打開,跟一般 App 一樣。

⚠️ 一定要用 Safari 打開這個網頁(從 LINE 點進來不行)。
  1. 用 Safari 打開網站
  2. 點畫面最下方中間的「分享」鈕(方框 + 向上箭頭)
  3. 往下滑,找到「加入主畫面」並點它
  4. 右上角點「加入」
  5. 回到桌面,就會看到 Istanda 圖示
⚠️ 用 Chrome 打開這個網頁(從 LINE 點進來請先改用 Chrome 開)。
  1. 用 Chrome 打開網站
  2. 點右上角「⋮」選單
  3. 選「安裝應用程式」或「加到主畫面」
  4. 點「安裝」/「加入」
  5. 回到桌面,就會看到 Istanda 圖示
